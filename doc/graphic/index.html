Zappy protocol for command and replies


The "Zappy" is a project made by Léo Thevenet, Clément Jean, Swann Polydor, Thibaut Lopez, Arnaud Lhomme and Romaric Fave.

It's a game where a various number of AIs connected to the Zappy server have to team up in order to collect food and minerals for leveling up their team.
